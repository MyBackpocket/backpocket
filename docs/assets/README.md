# Assets

Centralized logo and image assets for the Backpocket repository.

## Logos

| File | Size | Usage |
|------|------|-------|
| `Backpocket-Logo-32.png` | 32×32 | Inline with headings, favicons |
| `Backpocket-Logo-64.png` | 64×64 | Small icons, badges |
| `Backpocket-Logo-128.png` | 128×128 | Medium icons, thumbnails |
| `Backpocket-Logo-256.png` | 256×256 | Large icons, documentation |

## Source

These assets are copied from `apps/backpocket-web/assets/img/` which contains the full set of logos in multiple sizes and formats (PNG, WebP, AVIF).

## Usage in Markdown

```markdown
# <img src="./docs/assets/Backpocket-Logo-32.png" alt="Backpocket" height="32" align="absmiddle" /> Backpocket
```
